# Xbox Controller Shortcuts v0.1.0

First public release of Xbox Controller Shortcuts for macOS.

## Highlights

- Native macOS menu bar app
- Live button remapping from a settings window
- Support for keys, shortcuts, and scroll actions
- Built-in Accessibility status indicator
- Multilingual keyboard input normalization when possible

## Notes

- The app requires both `Accessibility` and `Input Monitoring` permission in macOS System Settings.
- If macOS blocks the app on first launch, use `Open` from Finder or right-click the app and choose `Open`.
- This build is not notarized yet, so Gatekeeper prompts may appear depending on your system settings.
